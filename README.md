# Sonara Frontend

Frontend de la plataforma Sonara para generación de audio con voces clonadas y creación de boletines informativos.

## Tecnologías

- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS + shadcn/ui
- React Hook Form + Zod
- @dnd-kit (drag & drop)
- date-fns, Sonner, Lucide

## Requisitos

- Node.js 18+
- pnpm (recomendado) o npm

## Instalación

```bash
git clone https://github.com/tu-usuario/sonara-frontend.git
cd sonara-frontend
pnpm install
```
